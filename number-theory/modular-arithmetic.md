# Modular Arithmetic – Basics

## Definition
For integers a, b and a positive integer n,
we say a ≡ b (mod n) if n divides (a − b).

## Example
17 ≡ 5 (mod 12) because 12 divides 12.

## Basic Properties
- a ≡ b (mod n) ⇒ a + c ≡ b + c (mod n)
- a ≡ b (mod n) ⇒ ac ≡ bc (mod n)

## Small Observation
If a ≡ b (mod n), then a^k ≡ b^k (mod n) for all k ≥ 1.

## Why this matters in Olympiad
Modular arithmetic is heavily used in number theory
problems to simplify large expressions.
